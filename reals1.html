Resolving Paradoxes in the Real Number Set $\Bbb{R}$

Let us motivate some analysis of the real number line and its properties. In this section, I aim to demonstrate the following:

  • Paradoxes in our real number line are numerous, clearly demonstrable, and require resolution using set theory
  • Extending our number system from the naturals through to the reals ($\Bbb{N} \subseteq \Bbb{Z} \subseteq \Bbb{Q} \subseteq \Bbb{R}$) is required if every length along the number line is to correspond to a value that we can actually define (Abbott, 2000 pg. 4).
  • Functions may be generalized to a definition that is much less strict than what is covered prior to Real Analysis. This is instrumental in Physics, as it allows for numerical techniques (via computer) to be carried out when analytical solutions do not exist, which is often.
  • Computational methods and resources have value even when approaching pure mathematics

Consider this bizarre function proposed by Dirichlet:

$$ f(x) = \begin{cases} 1\ \ \text{if} \ x \in \Bbb{Q} \\ 0\ \ \text{if} \ x \notin \Bbb{Q} \end{cases} \tag{Eq. 1}$$

Plotting this function over the real number line $\Bbb{R}$ produces a figure that prompts discussion about continuity, and more generally into the "fine structure" of $\Bbb{R}$. The plot conflicts with our limited definition of functions $(f)$ leading up to this point, as it cannot be expressed as an algebraic relationship and seems to test the borders of being single-valued.

Fig 1. - A plot of f(x). This is a single function, not two stacked on top of each other.

Here are some important observation about the properties of $f(x)$:

  • The irrational numbers (when $x \notin \Bbb{Q}$) are constructed from the "holes" in the rational number line above it, at $f(x)=1$. Is there a symmetry or some sort of pattern to this extreme discontinuity?
  • This is a loose way of visualizing the property that elements of $\Bbb{Q}$ are dense in $\Bbb{R}$
  • Paradoxically, both sets of points consist of infinitely-many elements, no matter our scope along $\Bbb{R}$. We must look into real analysis to compare of objects of infinite "size." This property of a number set is called cardinality.

An interesting way to show that $\Bbb{Q}$ is different from its subsets $\Bbb{N}$ and $\Bbb{Z}$ with regards to continuity is to create an analogue to Dirichlet's function with the integers. This time, the empty intervals between the integers of $\Bbb{Z}$ will be filled by the rationals of $\Bbb{Q}$:

$$ f_{\Bbb{Z}}(x) = \begin{cases} 1\ \ \text{if} \ x \in \Bbb{Z} \\ 0\ \ \text{if} \ x \notin \Bbb{Z} \end{cases} \tag{Eq. 2}$$
Fig 2. - A plot of $f_{\Bbb{Z}}(x)$ over a domain that is convenient for visualizing the empty intervals in $\Bbb{Z}$. These intervals are filled by densely-packed rationals.

This points towards the rational set being undoubtably "bigger" than the integer set. However, this is untrue. Cardinality and equinumerosity give us the result that $\Bbb{Z}$ and $\Bbb{Q}$ actually have the same number of elements.

  • In fact, it's even less intuitive. If |$\Bbb{Z}$| denotes the cardinatlity (# of elements) in the integer set $\Bbb{Z}$, then it is fact that:
$$ |\Bbb{N}|=|\Bbb{Z}|=|\Bbb{Q}|=\aleph_0 \tag{Eq. 3} $$ Using real analysis, we can arrive at this conclusion by showing that each set can be bijectively mapped onto any of the three.
