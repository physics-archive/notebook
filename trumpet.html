Convergence -- Infinite Series Can Return a Finite Value

Demonstrated here: the power of calculus in resolving problems created by infinity, and in obtaining analytical solutions to infinite sums. The following mathematical concept is convergence. Arrays (or series) of infinitely-many values will either a) diverge towards infinity or b) converge to an exact value.

Take a look at the following series: Expanded, this series has values:

$$\bbox[8px,border:1px solid black] { S_1 = \frac{1}{n^2}\,\ \,\text{where} \,\,n \in \Bbb{I}} \tag{Eq. 1}\label{s1}$$

We can expand this series to show that it is a very simple list of fractions.

$$\bbox[8px,border:1px solid black] { S_1=\frac{1}{n^2} = 1,\,\frac{1}{4},\,\frac{1}{9},\,\frac{1}{16},\,\frac{1}{25},\,\frac{1}{36},\,...} \tag{Eq. 1.2}\label{1.1}$$

Now consider the following sum over index $n \rightarrow \infty$:

$$\bbox[8px,border:1px solid black] { \sum_{n=1}^\infty = \frac{1}{n^2} = 1 + \frac{1}{4} + \frac{1}{16} + \frac{1}{25} +\,...\ } \tag{Eq. 2}\label{2}$$

In (Eq. 2), we are summing over index $n$ of infinite length. On the surface, we should be summing to infinity regardless of how small these additions become as $n$ grows. If an infinite sum will not return an infinite value, what value will it return and how do we even attempt a determination?

Drop the discrete notation, for now, in favor of a continuous function over domain $x \in [1,\infty)$. The discrete $S_1(n)$ becomes continuous:

$$\bbox[8px,border:1px solid black]{ f(x)=\int_{0}^{x}\frac{1}{x^2} \ dx} \tag{Eq. 3}\label{3}$$

Here is a plot of $x^{-2}$ with its underside shaded to represent $\int{x^{-2}dx}$. This is the "area-under-the-function" interpration for antiderivatives. This graphical interpretation can resolve a finite area under our infinite curve.

Fig. 1 - Plot of $1/x^2$ over domain $[1,4.5]$. This function monotonically decreases on $(0, \infty)$ such that $f(x \rightarrow \infty) \rightarrow 0$.

So, we need to integrate $f(x)$ over our domain $[1,\infty)$ and see what analytical solutions - if any - result.
Since $\infty$ is not itself a value, we should rewrite our indefinite integral using limits and a dummy variable $t$, and evaluate:

$$\bbox[8px,border:1px solid black]{ \lim_{t \rightarrow \infty} \int_{0}^{t}\frac{1}{x^2} = \lim_{t \rightarrow \infty} \left[1 - \frac{1}{t}\right] = 1 - \lim_{t \rightarrow \infty} \left[\frac{1}{t}\right] = \bbox[#FF1,8px,border:0px solid red]{1}\ } \tag{Eq. 4} $$

Eq. 4 - If $t \rightarrow \infty$, then the denomenator in $\frac{1}{t}$ approaches infinity. Therefore our quotient approaches zero -- and our integral is equal to 1.

Over the domain $[1,\infty)$, $f(x)$ encloses an area of exactly $1$. Determining that this infinite sum converges is not trivia; obtaining an analytical solution so easily by applying entry-level calculus is astonishing.



How can we demonstrate that this solution is not trivial?

Ask: if $\int_{1}^{\infty} \left[\frac{1}{x^2}\right]$ converges to a real value, then will $\int_{1}^{\infty} \left[\frac{1}{x}\right]$ also converge? Consider the similarities:

  • Both are rational expressions in the form $p/q$
  • Monotonically decrasing over $x \in [1,\infty)$
  • Has values in-between 0 and 1 over our domain
  • Mutual characteristics are graphically demonstrable:
Fig. 1 - Plot of $f_1(x)$ and $f_2(x)$ over the domain $[1,4.5]$. Visually evident: both functions are monotonic over this domain and approach 0 as $x \rightarrow \infty$. This graphical representation doesn't actually give us any information about convergence.

Let us evaluate our continuous sum for $f(x)=\left[\frac{1}{x}\right]$ and see if we diverge or converge:

$$\bbox[8px,border:1px solid black] { \lim_{t \rightarrow \infty} \int_{1}^{t} \left[\frac{1}{x}\right] = \lim_{t \rightarrow \infty} \ln[t]-\ln[1] = \infty - 0 = \bbox[#FF1,8px,border:0px solid red]{\infty} } \tag{Eq. 5}$$

Despite similarities between the functions, $f_1(x)$ converges over $[1,\infty)$ whereas $f_2(x)$ is divergent.

Evaluating function outputs over a continuous domain is less straight-forward, as a concept, than summing fractions in an infinite sequence (Eq. 1.2). Reason: integrating a function is far more abstract as a concept than adding a list of fractions together.

Therefore, let us return to our discrete series $S_1$ and attempt an evaluation using Fourier analysis. We can relate our continous function, $f(x)$, to our discrete infinite series $S_1(n)$.

  • Generally, Fourier analysis allows us to rewrite any function--with arbitrary precision--as an infinite sum of sines and cosines with coefficients. Convenient for periodic functions that are discontinuous over arbitrary period $L$.

$$\bbox[8px,border:1px solid black] { g(x) = \frac{a_0}{2} + \sum_{n=1}^{\infty} (a_n \cos(nx)+b_n sin(nx)) } \tag{Eq. 6}$$

The coefficients $a_0$, $a_n$ and $b_n$ are related to $g(x)$ via integration:

$$\bbox[8px,border:1px solid black] { a_0 = \frac{1}{2L} \int_{-L}^{L} g(x) dx } \bbox[8px,border:1px solid black] { a_n = \frac{1}{L} \int_{-L}^{L} g(x) cos\left[n \frac{x}{L}\right]dx } \bbox[8px,border:1px solid black] { b_n = \frac{1}{L} \int_{-L}^{L} g(x) sin\left[n \frac{x}{L}\right]dx } \tag{Eq.s 7-9}$$

Evaluating these three integrals yields one independent coefficient $(a_0)$ and two coefficients $(a_n, b_n)$ which vary with Fourier index $n$. The size of $n$ determines the precision of our series approximation of $g(x)$. Fourier analysis adapts the concept of convergence to fit approximation, computation and analytical purposes.

First introduced to me in Electrodyamics I: Parseval's theorem relates our (continuous, square-integrable) function to the fourier coefficients associated with $g(x)$ as such:

$$\bbox[8px,border:1px solid black] { \frac{1}{L} \int_{-L}^{L} \left[g^2(x)\right] dx = \frac{a^2_0}{2} + \sum_{n=1}^{\infty} \left[a^2_n+b^2_n\right] } \tag{Thrm. 1}$$

For certain choices for $g(x)$, Parseval's theorem builds on our ability with Fourier series to determine the value of convergence for infinite sum of discrete set $S_1$.

  • An advantage of (Thrm. 1) over (Eq. 6): our Fourier coefficients - $a_n$ and $b_n$ - now dominate their terms in the fourier series for $g(x)$. Obtaining an exact form for any $g(x)$ as an infinite series becomes much simpler.
