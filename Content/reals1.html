Resolving Paradoxes in the Real Number Set $\Bbb{R}$

Let us motivate some analysis of the real number line and its properties. In this section, I aim to demonstrate the following:

  • Set theory provides resolution to paradoxes in our real number system, such as infinities.
  • Extending our number system from the naturals through to the reals ($\Bbb{N} \subseteq \Bbb{Z} \subseteq \Bbb{Q} \subseteq \Bbb{R}$) is required if every length along the number line is to correspond to a value that we can actually define (Abbott, 2000 pg. 4).
  • Functions may be generalized to a definition that is much less strict than what is covered prior to real analysis. This is instrumental in Physics, as it allows for numerical techniques (via computer) to be carried out when analytical solutions do not exist, which is often.

The Holes in our Rational Number Line

Consider this bizarre function proposed by Dirichlet:

$$ f(x) = \begin{cases} 1\ \ \text{if} \ x \in \Bbb{Q} \\ 0\ \ \text{if} \ x \notin \Bbb{Q} \end{cases} \tag{Eq. 1}$$

Plotting this function on the reals produces a figure that prompts discussion about continuity. The plot conflicts with our limited definition of functions leading up to this point; it cannot be expressed as an algebraic relationship and seemingly pushes the borders of being single-valued.

Fig 1. - A plot of f(x). This is a single function, coded numerically in MATLAB; this is not a plot of two functions stacked on top of one another.

Here are some important observations about $f(x)$:

  • The irrational number line (at $f(x)=0$) is constructed from the "holes" in the rational number line above it (at $f(x)=1$).
  • This is a loose way of visualizing the property that elements of $\Bbb{Q}$ are dense in $\Bbb{R}$
  • Paradoxically, both sets of points consist of infinitely-many elements, no matter our scope along $\Bbb{R}$.

Providing Visualization for $\Bbb{Q}$'s Unique Density in $\Bbb{R}$

An interesting way to show that $\Bbb{Q}$ has density, as opposed to its subsets $\Bbb{N}$ and $\Bbb{Z}$, is to plot an analogue to Dirichlet's function, using the integers at $f(x)=1$ instead of the rationals. This time, the empty intervals between the upper set of points ($\Bbb{Z}$) will be filled by the rationals of $\Bbb{Q}$ -- analogous to (Fig. 1) where the holes of $\Bbb{Q}$ were filled by the irrationals:

$$ f_{\Bbb{Z}}(x) = \begin{cases} 1\ \ \text{if} \ x \in \Bbb{Z} \\ 0\ \ \text{if} \ x \in \Bbb{Q} \land x \notin \Bbb{Z} \end{cases} \tag{Eq. 2}$$
Fig 2. - A plot of $f_{\Bbb{Z}}(x)$ over a domain that demonstrates how empty intervals in $\Bbb{Z}$ are filled by densely-packed rationals. Notice that irrationals are omitted in (Eq. 2) from the second piecewise condition, yet the number set at $f_{\Bbb{Z}}(x)=0$ still looks like a continuum; it is not a continuum as it is missing the irrationals.

This property of $\Bbb{Q}$ to appear continuous, regardless of domain scope, is why the rational numbers form a field and support the identities for multiplication, division, addition and subtraction. For example, every element of $\Bbb{Z}$ has the multiplication inverses to support division. Still, we are missing many numbers in (Fig. 2) such as factors of $\pi$ or $\sqrt{2}$.

Boundedness of our Number System

The common starting point for characterizing our real number line is the Axiom of Completeness and a direct consequence -- the Archimedean Property. These are leveraged to define infinities, boundedness, and dense sets.

  • Axiom of Completeness: Every nonempty set of real numbers that is bounded above has a least upper bound (Abbott, 2000, p.14).
  • Archimedian Property (i): Given any number $x \in \Bbb{R}$, there exists an $n \in \Bbb{N}$ satisfying $n \gt x$
  • Archimedian Property (ii): Given any number $y \in \Bbb{R}$, there exists an $n \in \Bbb{N}$ satisfying $1/n \lt y$
     (Abbott, 2000, p.19)


Starting with the Axiom of Completeness: what does it mean for sets to be bounded, and how does this lead to resolution for paradoxes involving infinite sets?

  • Provides a generalization for minima and maxima, applied directly to sets.
  • Consequently, a rigorous definition for limits will eventually follow from this starting point.
Going back and making some observations about our number system:
  • The natural numbers set $\Bbb{N}$ leads with 1, so the natural set is bounded below (infinum of 1). It is not bounded above. Absence of an upper-bound fits historical notions of the infinite: the natural set begins but it never ends.
$$\Bbb{N} = \{1,2,3,4,\dots \}$$
  • The integer set $\Bbb{Z}$ is not bounded above, but unlike $\Bbb{N}$ the set is not bounded below. However, they are ordered and countably infinite, like the natural number set $\Bbb{N}$.
$$\Bbb{Z} = \{ \dots , -2, -1, 0, 1, 2, \dots \}$$
  • The rational field $\Bbb{Q}$ contains the set $\Bbb{Z}$. Therefore, $\Bbb{Q}$ will not be bounded below or above, yet it is still ordered and countably infinite.
$$\Bbb{Q} = \{ p/q : p \land q \in \Bbb{Z},\ q \notin 0 \}$$
Interestingly, it is clear that defining $\Bbb{Q}$ requires "more" notation; the Archimedean Property guarantees that the rationals are packed "infinitesimally" close to one-another, based on the above definition.

Despite being countably infinite like the naturals and integers, it is tricker to list adjacent terms.

Applying this concept elsewhere, consider some set $A$ with elements that are ordered as such:

$$ A = \{r^2 \gt 2\ :\ x \in \Bbb{Q}\} \tag{Eq. 3} $$

Set $A$ contains all rational numbers $r^2$ that are greater than 2, not including 2 itself.

  • Rational numbers form an ordered set (field) that is countable, and set A contains all rationals greater than 2
  • Therefore, set A is also ordered and countable; these properties are inherited from $\Bbb{Q}$.
  • A must be bounded below with a greatest lower bound (infinum) of 2, despite $2 \notin A$.
$$ \bbox[8px,border:1px solid black]{ \text{inf} A=2} $$
