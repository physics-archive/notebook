True Randomness in Nature

A reasonable question that is frequently asked of physicists: is anything in nature truly random? The answer is yes.

  • Many events which produce random-seeming outcomes are not actually random (i.e. coin tosses).
  • If an experimentalist were to replicate exactly a system configuration which resulted in tails, they could reproduce that result ad infinitum.
  • However, the outcome of a completely-random event cannot be determined before the event occurs.
Indeed, classical mechanics is a deterministic model of the universe. There are no truly random events that occur.
